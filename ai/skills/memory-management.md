# Skill — Memory Management

## Objectif

Maintenir une mémoire projet cohérente, utile et exploitable dans le temps.

## Règles

- la mémoire doit refléter le comportement validé réel
- distinguer contexte stable et historique vivant
- documenter les décisions structurantes
- éviter les duplications inutiles
- conserver les invariants importants
- expliciter les limitations connues

## Refuser si

- la mémoire diverge du code réel
- des hypothèses non validées sont mémorisées
- des décisions importantes disparaissent
