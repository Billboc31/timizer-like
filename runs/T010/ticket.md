# T010 — Bootstrap React frontend

**Source**: GitHub Issue #3

## Description

## Context

The MVP requires a simple web interface to enter monthly CRA days and download generated PDFs.

## Goal

Create the frontend application foundation.

## Description

Initialize a React and Vite frontend project inside the repository with a clear frontend folder structure.

The frontend must be runnable locally and display a basic application shell.

## Out of Scope

- CRA calendar
- API integration
- PDF download
- Styling polish
- Authentication
- Backend code

## Acceptance Criteria

- Frontend project exists in the repository
- Frontend can start locally
- A basic application shell is displayed
- Frontend dependencies are declared in the project configuration
- Existing project files are not broken
